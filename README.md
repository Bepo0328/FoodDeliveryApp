# FoodDeliveryApp
음식 배달 앱

## ⚠️ 주의사항
- [Firebase 프로젝트](https://console.firebase.google.com/u/0/) 에서 다운받은 `google-services.json` 파일을 추가  필요.
- `gradle.properties` 파일에 `TMAP API 키` 를 추가 필요.
- `api-key.xml` 파일에 `KaKao API 키` 를 추가 필요.
